# Karim Benkhira - Cybersecurity Portfolio

A modern, interactive portfolio website showcasing cybersecurity expertise, certifications, and experience.

## Features

- **Matrix Rain Effect**: Animated background with falling characters
- **Glitch Effects**: Cyberpunk-style text animations
- **Typing Animation**: Dynamic role display
- **Smooth Scrolling**: Seamless navigation between sections
- **Interactive Stats**: Animated counters and progress bars
- **Responsive Design**: Mobile-friendly layout
- **Particle Effects**: Mouse tracking visual effects

## Technologies Used

- HTML5
- CSS3 (with custom animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Orbitron & Rajdhani)

## Sections

1. **Hero**: Introduction with animated typing effect
2. **About**: Terminal-style biography with statistics
3. **Experience**: Timeline of professional roles
4. **Certifications**: Showcase of security credentials
5. **Skills**: Technical and soft skills with progress bars
6. **Contact**: Get in touch information

## Setup

1. Clone or download this repository
2. Add your CV PDF file to the `assets` folder and name it `Karim_Benkhira_CV.pdf`
3. Open `index.html` in your web browser
4. No build process required - pure HTML/CSS/JS

## File Structure

```
portfolio/
├── index.html          # Main HTML file
├── styles.css          # Styling and animations
├── script.js           # Interactive features
├── assets/             # Assets folder
│   └── Karim_Benkhira_CV.pdf  # Your CV (add this file)
└── README.md           # Documentation
```

## Customization

- Edit content in `index.html`
- Modify colors and styles in `styles.css`
- Adjust animations and effects in `script.js`
