# CV File Instructions

Please add your CV PDF file to this folder with the name: `Karim_Benkhira_CV.pdf`

The portfolio website is configured to download this file when users click the "Download CV" button.

## Steps:
1. Save your CV as a PDF file
2. Rename it to: `Karim_Benkhira_CV.pdf`
3. Place it in this `assets` folder

The download button can be found in:
- Navigation bar (top right)
- Hero section (main buttons)

If you want to change the CV filename, update these locations in `index.html`:
- Line with `assets/Karim_Benkhira_CV.pdf` in navigation
- Line with `assets/Karim_Benkhira_CV.pdf` in hero buttons
